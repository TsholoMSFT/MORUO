M.O.R.U.O is a React + Vite app for evaluating business value scenarios, with an optional Azure Functions backend for AI chat/recommendations.

## Development
- Frontend: `npm install` then `npm run dev`
- API (Azure Functions): see the `api/` folder

## Theme
The app theme is dark-only and defined in `src/main.css` using OKLCH CSS variables that Tailwind tokens map to.
